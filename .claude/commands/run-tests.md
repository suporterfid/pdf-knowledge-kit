# Run Tests

Execute the full test suite for both backend (Python) and frontend (TypeScript).

## Steps

1. Run backend tests with pytest
   - Execute: `pytest -v`
   - Show coverage report if available

2. Run frontend tests with vitest
   - Change to frontend directory
   - Execute: `npm test`
   - Show test results summary

3. Report any failures clearly with file paths and line numbers

Display test results in a clear, organized format showing passing, failing, and skipped tests.
